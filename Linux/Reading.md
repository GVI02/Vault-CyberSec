How to read files in Linux CLI.

Read contents of a file and output it to the console using **cat**.
```shell
cat file.txt
```

Read first 10 lines of a file using **head**.
```shell
head file.txt
```

Read last 10 lines of a file using **tail**.
```shell
tail file.txt
```
